# 股票庫藏追蹤系統（每日自動更新）

## 🎯 需求
建立一個網頁：

- 每天自動更新（無需手動）
- 使用 Python 計算股票庫藏績效
- 支援台股 / 美股
- 顯示：
  - 損益（$ / %）
  - 時間區間（週 / 月 / 年 / 長期）
  - 配息
  - 多帳戶
- 部署在 GitHub Pages

---

# 🧠 核心問題

👉 Python 要在哪裡執行？

## ✅ 結論
有兩種方式：

### 1️⃣ GitHub Actions（推薦）
👉 在 GitHub 雲端執行

### 2️⃣ 本機 PC
👉 用排程工具執行（不推薦）

---

# 🟢 方案一：GitHub Actions（推薦）

## 🔁 流程

```text
每天定時
↓
GitHub Actions 啟動
↓
執行 Python 腳本
↓
更新 JSON / HTML
↓
commit 回 repo
↓
GitHub Pages 自動更新
```

---

## 已加入的部署功能

本專案已新增 GitHub Actions workflow：

```text
.github/workflows/stock-treasury-pages.yml
```

功能：

- 手動執行：GitHub repo → Actions → Stock Treasury Pages → Run workflow
- 排程執行：每天台灣時間 `13:45` 自動更新
- 自動安裝 Python 3.11 與 requirements
- 自動執行 `app.py`
- 產出 `stock_treasury_value_tracker/output/report.html`
- 產出 `stock_treasury_value_tracker/output/report.json`
- 自動 commit 產出的 report 回 repo
- 自動部署到 GitHub Pages

手動執行時可以填 `reference_ticker_twd`，例如：

```text
0050.TW
006208.TW
```

GitHub Pages 設定：

1. 到 GitHub repo 的 `Settings`
2. 點 `Pages`
3. `Build and deployment` 選 `GitHub Actions`
4. 到 `Actions` 手動執行 `Stock Treasury Pages`

本機也可以產出 HTML + JSON：

```powershell
Set-Location "d:\One-Drive\OneDrive - Realtek Semiconductor Corp\Python\stock_treasury_value_tracker"
py -3.11 app.py --cache-hours 999999 --reference-ticker-twd 0050.TW --output output\report.html --json-output output\report.json
```

### 資料更新流程

要改庫存資料時，改 PC 端這兩個檔：

```text
data/transactions.csv
data/dividends.csv
```

建議流程：

```powershell
Set-Location "d:\One-Drive\OneDrive - Realtek Semiconductor Corp\Python\stock_treasury_value_tracker"
git pull
# 修改 CSV
git add data/transactions.csv data/dividends.csv
git commit -m "Update portfolio data"
git push
```

push 後：

- GitHub 端會因為 `push` trigger 自動重建報表與 Pages
- 不需要等到 13:45

### 同步規則

- `PC 改程式碼/CSV -> git commit -> git push -> GitHub 自動更新`
- `GitHub Actions 自動產生新的 output/report.html / report.json -> 不會自動回到 PC`
- 要把 GitHub 端最新報表同步回 PC，需要：

```powershell
git pull
```

### 手動部署方式
